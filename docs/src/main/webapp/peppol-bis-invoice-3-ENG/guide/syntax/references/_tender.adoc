
= Tender reference (CIG)

To identify the call for tender or lot the invoice relates to, use the 'OriginatorDocumentReference'. The identifier is, in most cases, the Procurement Procedure Identifier.

#The CIG code is mandatory in contexts where it is required, and is necessary for invoice verification, reconciliation and payment approval processes.#


.UBL example of tender reference (CIG)
[source, xml, indent=0]
----
<cac:OriginatorDocumentReference>
    <cbc:ID>90305884F7</cbc:ID>
</cac:OriginatorDocumentReference>
----

#To refer to multiple CIG codes from invoice lines, use *cac:DocumentReference* with document type code = 130.#

.UBL example of tender reference (CIG) from invoice lines
[source, xml, indent=0]
----
<cac:DocumentReference>
    <cbc:ID schemeID="AGB">90305884F7</cbc:ID>
    <cbc:DocumentTypeCode>130</cbc:DocumentTypeCode>
</cac:DocumentReference>
----
