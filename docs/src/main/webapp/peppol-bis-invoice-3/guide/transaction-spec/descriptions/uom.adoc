= Unità di misura

L'unità di misura in una fattura prevede l'utilizzo delle codifiche presenti in UNECE Recommendation No. 20 (version 11e), unitamente alle codifiche riportate in UNECE Recommendation No. 21 con il prefisso X. 
Si consiglia di consultare <<uom>> per ulteriori dettagli riferiti alle codifiche di riferimento.
