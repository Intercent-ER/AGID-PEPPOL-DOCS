
= #Regime Fiscale# (estensione)

Il regime fiscale indica il regime contabile associato al fornitore e pertanto le agevolazioni o semplificazioni a cui Ã¨ soggetto.

Se il regime fiscale non viene specificato, si intende il regime ordinario (RF01).

[source, xml, indent=0]

<cac:AccountingSupplierParty>
    <cac:Party>

            <cbc:CompanyID>IT07643520567</cbc:CompanyID>
            <cbc:TaxLevelCode>RF19</cbc:TaxLevelCode>
            <cac:TaxScheme>
                <cbc:ID>VAT</cbc:ID>
            </cac:TaxScheme>
        </cac:PartyTaxScheme>
    ...
    </cac:Party>
</cac:AccountingSupplierParty>
----
