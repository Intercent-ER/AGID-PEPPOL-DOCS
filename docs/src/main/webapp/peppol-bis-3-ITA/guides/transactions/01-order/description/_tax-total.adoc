[[tax-total]]
= Totale imposte

E’ possibile fornire il totale delle imposte dell’ordine sia livello di testata (tax total) che a livello di riga (taxCategory and rate).


.Livello di testata:
[source, xml, indent=0]
----
<cac:TaxTotal>
    <cbc:TaxAmount currencyID="EUR">5.00</cbc:TaxAmount>
</cac:TaxTotal>
----
