[[classification]]
= Classificazione

Un artioclo potrebbe essere classificato in accordo alla tassonomia *UNSPSC* (United Nations Standard Products and Services Code), la quale rappresenta la classificazione pubblica obbligatoria degli schemi in alcuni paesi/settori. +

Considerato che non è presente nessun codice per la UNSPSC al momento nella codelist UNCL 7143, si può usare il codice "MP" (Numero di identificazione bene/servizio). +

I prodotti posso essere inoltre classificati secondo gli schemi regolatori o classificatori usati in alcuni settori di business.
