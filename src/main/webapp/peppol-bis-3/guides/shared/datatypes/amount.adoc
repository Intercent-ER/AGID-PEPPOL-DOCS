= Amount

Un importo indica un valore numerico monetario. La valuta dell’importo è definito come un termine finanziario distinto.

====
IMPORTANT: L’importo viene espresso come numero con al più due cifre decimali.
====

|===
|Componente
|Uso
|Tipo primitivo
|Esempio

|Contenuto
|Obbligatorio
|Decimal
|10000.25
|===
