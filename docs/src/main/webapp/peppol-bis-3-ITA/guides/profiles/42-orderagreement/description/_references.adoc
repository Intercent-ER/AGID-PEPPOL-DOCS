= Riferimenti

Durante l'invio di un ordine pre-concordato un Venditore può includere un riferimento chel'Acquirente gli ha fornito durante l'acquisto. Questo riferimento può essere di diversa natura e considerato che è originato dall'Acquirente, è lui il soggetto che lo capisce.
