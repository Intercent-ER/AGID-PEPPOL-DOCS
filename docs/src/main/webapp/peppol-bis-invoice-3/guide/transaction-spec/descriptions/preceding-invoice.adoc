
= Riferimento a fatture precedenti

Una nota di credito o fattura negativa può fare riferimento ad una o più fatture precedenti. Questo è possibile nel raggruppamento BG-3 Preceding invoice reference, fornendo il numero e la data fattura. La data fattura deve essere fornita nel caso il numero di fattura precedente in riferimento non è univoco.

Nel caso che la correzione si applichi ad un largo numero di fatture, il periodo di fatturazione (BG-14), combinato opportunamente ad una nota esplicativa (BT-22), può alternativamente essere fornito in testata.
