
= #Indirizzo (CIUS)#

[lime-background]#L'utilizzo dell’aggregato informativo cac:PostalAddress è diventato obbligatorio, ma i requisiti dettagliati di questo componente vengono mantenuti opzionali, perché questi dipendono sia da regole EU che nazionali. Sotto si riportano due esempi di come compilare le informazioni dell’indirizzo.#  

.Indirizzo semplice
[source, xml]
<cac:PostalAddress>
	<cbc:StreetName>Via principale 2, Palazzo 4</cbc:StreetName>
	<cbc:CityName>Grande città</cbc:CityName>
	<cbc:PostalZone>54321</cbc:PostalZone>
	<cac:Country>
		<cbc:IdentificationCode>IT</cbc:IdentificationCode>
	</cac:Country>
</cac:PostalAddress>

.Indirizzo completo
[source, xml]
<cac:PostalAddress>
	<cbc:StreetName>Corso di Francia</cbc:StreetName>
	<cbc:AdditionalStreetName>I Parioli</cbc:AdditionalStreetName>
	<cbc:CityName>Roma</cbc:CityName>
	<cbc:PostalZone>00197</cbc:PostalZone>
	<cbc:CountrySubentity>RM</cbc:CountrySubentity>
	<cac:AddressLine>
		<cbc:Line>Informazioni Aggiuntive</cbc:Line>
	</cac:AddressLine>
	<cac:Country>
		<cbc:IdentificationCode>IT</cbc:IdentificationCode>
	</cac:Country>
</cac:PostalAddress>
